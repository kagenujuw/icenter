# pc-mockjs

> A Vue.js project

##describle
本项目，主要采用的技术站vue、vue-router、vuex、element-ui、mockjs、axios进行单页应用的开发
plugins目录自己封装了alert、confirm、dialog等插件
本地通过mockjs，来模拟假数据进行开发

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# start mockServer at localhost:3000
npm run mock 

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
